EPI Memory

EPI Memory est né d'un constat simple: quand une nouvelle promo arrive à EPITECH, il est difficile de se souvenir tout de suite du login de tout le monde.
Pour ne pas avoir à demander à chaque fois aux étudiants leur login, il faut les mémoriser... Et pour cela, quoi de mieux qu'un jeu ?

EPI Memory est écrit en javascript et utilise l'excellent framework angularjs. Un petit backend en php permet de récupérer les données de manière authentifiée, afin de ne pas rendre publiques les infos aspirées sur l'intra (liste des logins).
